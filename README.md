# 🎯 Number Guessing Game (C++)

## Overview

This project is a simple console-based Number Guessing Game developed in C++. The program generates a random number between 1 and 100, and the player attempts to guess it. After each guess, the program provides feedback indicating whether the guess is too high or too low until the correct number is found.

## Features

- Random number generation
- Unlimited guessing attempts
- Feedback after each guess
- Tracks the number of attempts
- Play again option
- Console-based user interface

## Technologies Used

- C++
- Visual Studio 2022
- Standard C++ Library

## Skills Demonstrated

- Random number generation (`rand()` and `srand()`)
- Loops (`do...while`)
- Conditional statements (`if` / `else`)
- Variables and data types
- User input and output
- Program flow control

## How to Run

1. Clone the repository.
2. Open the project in Visual Studio.
3. Build and run the application.
4. Guess the randomly generated number.
5. Continue guessing until you win.
6. Choose whether to play another round.
